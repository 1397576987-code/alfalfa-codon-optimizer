# Alfalfa Codon Optimizer

一个面向紫花苜蓿及苜蓿属材料的密码子优化网页工具。用户可以输入蛋白序列或 CDS 序列，选择不同苜蓿材料的密码子偏好预设，生成适合目标材料表达的优化 CDS 序列。

## 功能特点

- 支持蛋白序列和 CDS 序列输入
- 自动识别 FASTA 格式并忽略标题行
- 支持多个苜蓿材料/品种预设
- 输出优化后的 CDS FASTA 序列
- 计算 GC 含量和 CAI-like 评分
- 显示序列长度、氨基酸数量和替换密码子数量
- 标记常见限制性内切酶位点
- 支持一键复制和下载 FASTA
- 单文件 HTML 版本，可直接双击打开或部署到 GitHub Pages

## 支持的密码子偏好预设

本工具的品种预设基于《苜蓿基因组密码子偏好性分析》中不同苜蓿材料的最优密码子结果设计，包括：

- 苜蓿通用共识
- 新疆大叶
- 中苜1号
- 中苜4号
- 蒺藜苜蓿 A17

其中，“苜蓿通用共识”综合了多个材料中较稳定的偏好密码子；各品种预设则优先使用对应材料的最优密码子。

## 使用方法

1. 打开网页。
2. 在输入框中粘贴蛋白序列或 CDS 序列。
3. 选择目标品种预设。
4. 选择优化模式：
   - 平衡优化：优先使用品种最优密码子，同时兼顾 GC 含量
   - 最高频密码子：优先使用该品种对应氨基酸的首选最优密码子
   - GC 目标优先：在密码子偏好基础上更强调目标 GC%
5. 点击“优化序列”。
6. 复制或下载优化后的 FASTA 序列。

## 数据来源

本工具的品种密码子优化选项参考以下研究中的最优密码子分析结果：

李琳, 徐明志, 刘燕蓉, 张万军. 苜蓿基因组密码子偏好性分析. 草地学报, 2024, 32(9): 2695-2706.

该研究分析了蒺藜苜蓿、新疆大叶、中苜1号和中苜4号的基因编码区密码子组成、RSCU、ENC、GC3 等指标，并确定了不同材料的最优密码子。

## 注意事项

- 本工具用于密码子优化设计和初步筛选，不替代实验验证。
- 正式基因合成前，建议结合载体要求、限制性酶切位点、重复序列、RNA 二级结构、剪接位点、甲基化风险和目标表达系统进行进一步检查。
- CAI-like 评分是工具内部根据当前预设计算的相对适配指标，适合不同优化方案之间比较，不等同于严格的实验表达量预测。
